End-to-end data pipeline built on Databricks processing NYC Yellow Taxi trip records. Covers ingestion, schema enforcement, distributed aggregation, and behavioral analysis across millions of trips.

---

# What I Did

- Schema Enforcement & Type Casting
- Distributed Aggregation with PySpark
- Derived Metric Engineering
- Multi-Key Filtering & Ranking
- Modular Pipeline Design

# Dataset

**NYC TLC Yellow Taxi Trip Records** — includes pickup/dropoff datetimes, passenger counts, trip distances, fare breakdowns, and location IDs.

> Source: [NYC Taxi & Limousine Commission](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page)
